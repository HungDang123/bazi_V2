# Thư mục nguồn import

Đặt file Excel/DOCX dùng cho các lệnh `php artisan import:*` tại đây.

Ví dụ:

```bash
php artisan import:phan6-xlsx
php artisan import:phan5-khia-canh --fresh
php artisan import:nhat-chu-tru-ngay
```

Có thể truyền đường dẫn tùy chỉnh: `php artisan import:bo-cuc-ngu-hanh imports/bo_cuc_ngu_hanh.xlsx`
